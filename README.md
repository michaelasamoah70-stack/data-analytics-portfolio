## Project 1 — Cyclistic Bike-Share Analysis

### Overview
An end-to-end data analysis project completed as the Google Data Analytics 
Certificate capstone. The project analyses 124,475 trip records to understand 
how annual members and casual riders use Cyclistic bikes differently.

### Business Task
How do annual members and casual riders use Cyclistic bikes differently — 
and what data-driven recommendations can convert casual riders into 
annual members?

### Tools Used
- MySQL — data cleaning and transformation
- Jupyter Notebook — SQL execution and documentation
- Tableau Public — data visualisation and dashboard

### Key Findings
1. Members account for 82.61% of all rides — casual riders only 17.39%
2. Casual riders take longer trips on average — 12.99 mins vs 11.19 mins
3. Members show commuter peaks at 8AM and 5PM — casual riders peak in the afternoon
4. Casual riders prefer electric bikes more strongly — 73.10% vs 63.10%
5. Members ride consistently all week — casual riders take longest rides on weekends
